Smart Pattern Management
Organize complex productions by splitting patterns into individual channels

Split by Channel:

  1. Name all plugins first (Shift + Left Click)
  2. Route all plugins (Ctrl + L)
  3. Right-click pattern and select "Split by Channel"
  4. Now arrange individual elements on playlist with perfect organization

Multi-Arrangement Workflow
Work on multiple versions of your track simultaneously

Use multiple arrangements with dedicated pattern groups:

  1. Create pattern groups (Right click in browser)
  2. Use one arrangement as "sketchpad" for experimentation
  3. Keep main arrangement clean for final production
  4. Filter patterns by group for better organization

Tip: Use Alt + drag to clone entire pattern sections between arrangements
